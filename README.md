# 🧠 GlucoBot — AI Diabetes Risk Assistant

GlucoBot is an AI-powered diabetes risk prediction and health guidance web app.

## 🚀 Features

- Machine Learning based diabetes risk prediction  
- Animated AI risk gauge visualization  
- Smart conversational diabetes assistant  
- Medical report PDF upload & auto-fill  
- Downloadable printable AI health report  
- Preventive lifestyle guidance  

## 🛠 Tech Stack

- Python  
- Streamlit  
- Scikit-learn  
- Matplotlib  
- ReportLab  

## ▶️ Run Locally

pip install -r requirements.txt
streamlit run app.py
